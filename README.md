# img2txt
Convert  GIF, JP(E)G, or PNG image to text. Not very useful, but pretty cool.

# quick start

``` go
img2txt := NewImg2txt()
img2txt.SetPath("./look.png").SetStep(2).SetScale(3).PrintTxt()
```
